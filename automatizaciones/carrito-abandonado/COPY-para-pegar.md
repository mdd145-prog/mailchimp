# Carrito Abandonado — Textos para pegar en Mailchimp

> Copiá y pegá cada bloque en el editor de Mailchimp. El **bloque de productos** del carrito lo agrega Mailchimp solo (bloque "Producto" / cart, se llena con lo que el cliente dejó). Vos solo pegás los textos.

---

## ✉️ EMAIL 1 — a la 1 hora

**Asunto:**
```
Dejaste botellas en el carrito
```
**Preheader (texto de preview):**
```
Las apartamos por vos. Siguen disponibles.
```
**Título (arriba):**
```
Quedaron botellas esperando
```
**Texto:**
```
Las apartamos por vos, pero el stock es el que es. Retomá tu pedido cuando quieras.
```
→ acá va el **bloque de productos del carrito** (lo agrega Mailchimp)

**Botón:**
```
RETOMAR MI PEDIDO
```
**Pie:**
```
¿Una consulta? Respondé este mail o escribinos por WhatsApp.
```


> **Nota para whisky/espirituosas/guardados** (carritos sin 6×5): en el Email 3, en vez del recuadro del 6×5, ofrecé **3 cuotas sin interés** o **envío sin cargo**. Nunca un % de descuento salvo que sea la única forma de no perder la venta.

## Reglas de oro (ya están pensadas en los textos)
- Email 1 y 2: **sin descuento**. El 80% se recupera sin regalar margen.
- Email 3 (vinos): el "incentivo" es **recordar el 6×5 que ya existe**, no uno nuevo.
- Quien está en este recorrido **no debería recibir el newsletter** ese día.
